# Epic Online Services Godot (EOSG)

Epic Online Services Godot (EOSG) is the easiest way to integrate Epic Online Services into your Godot 4.2+ projects. This plugin wraps the Epic Online Services C SDK using GDExtension, making it seamlessly accessible through GDScript, C#, and other Godot-supported languages.

## Two Ways to Use EOS

This plugin provides two interfaces for working with Epic Online Services:

### High Level EOS (Recommended)

The High Level Epic Online Services provides easy-to-use methods and signals to interact with EOS, making it ideal for beginners. This simplified API abstracts away many of the complexities of the EOS SDK while providing access to its core functionality.

**Main Classes:**
- `HPlatform` - Platform initialization and management
- `HAuth` - User authentication and login flows
- `HAchievements` - Achievement system
- `HFriends` - Friends list and social features
- `HStats` - Player statistics
- `HLeaderboards` - Leaderboard management
- `HLobbies` - Lobby and matchmaking
- `HP2P` - Peer-to-peer networking
- `HLog` - Logging utilities

### GDExtension EOS

The GDExtension EOS provides low-level access to the Epic Online Services SDK. This interface is not recommended for beginners but gives you full access to the underlying EOS functionality through the `EOS` and `IEOS` classes in Godot.

## Quick Example

Here's a basic example of using the High Level EOS API to initialize the platform and authenticate a user:

```gdscript Main.gd
extends Node

func _ready() -> void:
    # Setup HEOS Logs
    HLog.log_level = HLog.LogLevel.INFO

    var credentials = HCredentials.new()
    credentials.product_name = "My Awesome Game"
    credentials.product_version = "1.0"
    credentials.product_id = "YOUR_PRODUCT_ID"
    credentials.sandbox_id = "YOUR_SANDBOX_ID"
    credentials.deployment_id = "YOUR_DEPLOYMENT_ID"
    credentials.client_id = "YOUR_CLIENT_ID"
    credentials.client_secret = "YOUR_CLIENT_SECRET"

    var setup_success := await HPlatform.setup_eos_async(credentials)
    if not setup_success:
        printerr("Failed to setup EOS. See logs for more details")
        return

    HAuth.logged_in.connect(_on_logged_in)
    
    # Login anonymously for testing
    await HAuth.login_anonymous_async()

func _on_logged_in():
    print("Logged in successfully: product_user_id=%s" % HAuth.product_user_id)
    
    # Example: Get top records for a leaderboard
    var records := await HLeaderboards.get_leaderboard_records_async("YOUR_LEADERBOARD_ID")
    print(records)
```
