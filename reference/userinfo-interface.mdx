The UserInfo Interface retrieves user profile information including display names and external account data.

## Overview

The UserInfo Interface provides:

- Querying user display names
- Retrieving external account information
- Getting best display names for users
- Platform-specific display name handling

All methods are called on the `IEOS` singleton instance and use Epic Account IDs.

## Methods

### user_info_interface_query_user_info

```gdscript
IEOS.user_info_interface_query_user_info(options: RefCounted) -> void
```

Queries user information from the backend.

**Parameters:**
- `options` - Dictionary containing:
  - `local_user_id` - Epic Account ID making the request
  - `target_user_id` - Epic Account ID to query
  - `client_data` - Optional user data

**Callback Signal:** `user_info_interface_query_user_info_callback`

**Callback Parameters:**
- `result_code` - EOS_EResult
- `local_user_id` - Epic Account ID
- `target_user_id` - Epic Account ID

## Example Usage

```gdscript
# Query user info
var query_options = {
    "local_user_id": epic_account_id,
    "target_user_id": friend_epic_id
}

IEOS.user_info_interface_query_user_info_callback.connect(_on_user_info_queried)
IEOS.user_info_interface_query_user_info(query_options)

func _on_user_info_queried(data: Dictionary):
    if data.result_code == EOS_EResult.EOS_Success:
        var user_info = IEOS.user_info_interface_copy_user_info({
            "local_user_id": epic_account_id,
            "target_user_id": friend_epic_id
        })
        print("User info: ", user_info.user_info)

# Get best display name
var display_name = IEOS.user_info_interface_copy_best_display_name({
    "local_user_id": epic_account_id,
    "target_user_id": friend_epic_id
})

if display_name.result_code == EOS_EResult.EOS_Success:
    print("Display name: ", display_name.best_display_name)
```
