The Sessions Interface manages multiplayer game sessions with more advanced features than lobbies.

## Overview

The Sessions Interface provides:

- Session creation and management
- Session search and discovery
- Player registration for matchmaking
- Session invites
- Presence integration
- Platform crossplay controls

All methods are called on the `IEOS` singleton instance.

## Methods

### sessions_interface_create_session_modification

```gdscript
IEOS.sessions_interface_create_session_modification(options: RefCounted) -> Dictionary
```

Creates a session modification handle for a new session.

**Parameters:**
- `options` - Dictionary containing:
  - `session_name` - Session name string
  - `bucket_id` - Bucket ID for matchmaking
  - `max_players` - Maximum player count
  - `local_user_id` - Product User ID (optional)
  - `presence_enabled` - Boolean
  - `session_id` - Optional custom session ID
  - `sanctions_enabled` - Boolean
  - `allowed_platform_ids` - Optional array of platform IDs

**Returns:** Dictionary with `result_code` and `session_modification`

### sessions_interface_update_session_modification

```gdscript
IEOS.sessions_interface_update_session_modification(options: RefCounted) -> Dictionary
```


- `sanctioned_players` - Array of sanctioned Product User IDs

## Example Usage

```gdscript
# Create a session
var mod_result = IEOS.sessions_interface_create_session_modification({
    "session_name": "MySession",
    "bucket_id": "MyGame:Region1",
    "max_players": 8,
    "presence_enabled": true
})

if mod_result.result_code == EOS_EResult.EOS_Success:
    var session_mod = mod_result.session_modification
    # Configure session attributes on session_mod...
    
    IEOS.sessions_interface_update_session_callback.connect(_on_session_updated)
    IEOS.sessions_interface_update_session({"session_modification": session_mod})

func _on_session_updated(data: Dictionary):
    if data.result_code == EOS_EResult.EOS_Success:
        print("Session created: ", data.session_name)
```
