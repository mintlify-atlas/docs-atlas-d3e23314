## Overview

The `HPlatform` class handles the initialization and creation of the EOS platform. It manages SDK initialization, platform creation, and logging configuration. This is typically the first HEOS class you'll interact with when setting up Epic Online Services.

## Signals

### platform_initialized

Emitted when the EOS platform was initialized successfully.

```gdscript
signal platform_initialized
```

### platform_created

Emitted when the EOS platform was created successfully.

```gdscript
signal platform_created
```

### log_msg

Emitted when the EOS SDK produces a log message. Use this signal to handle the log however you want.

```gdscript
signal log_msg(msg: EOS.Logging.LogMessage)
```

<ParamField path="msg" type="EOS.Logging.LogMessage">
  The log message from the EOS SDK
</ParamField>

## Methods

### setup_eos_async

```gdscript
func setup_eos_async(p_creds: HCredentials) -> bool
```

Easy setup for EOS. Returns true if EOS setup is success.

<ParamField path="p_creds" type="HCredentials" required>
  Credentials for Epic Online Services
</ParamField>

<ResponseField name="return" type="bool">
  Returns `true` if EOS setup was successful, `false` otherwise
</ResponseField>

**Example:**

```gdscript
var creds = HCredentials.new()
creds.product_name = "My Game"
creds.product_version = "1.0"
creds.product_id = "your_product_id"
creds.sandbox_id = "your_sandbox_id"
creds.deployment_id = "your_deployment_id"
creds.client_id = "your_client_id"
creds.client_secret = "your_client_secret"

var success = await HPlatform.setup_eos_async(creds)
if success:
    print("EOS initialized successfully")
```

### initialize_async

```gdscript
func initialize_async(opts: EOS.Platform.InitializeOptions) -> EOS.Result
```

Initialize the EOS SDK.

<ParamField path="opts" type="EOS.Platform.InitializeOptions" required>
  Initialization options for the EOS SDK
</ParamField>

<ResponseField name="return" type="EOS.Result">
  Result code from the initialization attempt
</ResponseField>

**Example:**

```gdscript
var init_opts = EOS.Platform.InitializeOptions.new()
init_opts.product_name = "My Game"
init_opts.product_version = "1.0"

var result = await HPlatform.initialize_async(init_opts)
if EOS.is_success(result):
    print("SDK initialized")
```

### create_platform_async

```gdscript
func create_platform_async(opts: EOS.Platform.CreateOptions) -> bool
```

Create the EOS Platform.

<ParamField path="opts" type="EOS.Platform.CreateOptions" required>
  Options for creating the EOS platform
</ParamField>

<ResponseField name="return" type="bool">
  Returns `true` if platform was created successfully, `false` otherwise
</ResponseField>

**Example:**

```gdscript
var create_opts = EOS.Platform.CreateOptions.new()
create_opts.product_id = "your_product_id"
create_opts.sandbox_id = "your_sandbox_id"
create_opts.deployment_id = "your_deployment_id"
create_opts.client_id = "your_client_id"
create_opts.client_secret = "your_client_secret"

var success = await HPlatform.create_platform_async(create_opts)
if success:
    print("Platform created")
```
