## Overview

The `HLeaderboards` class provides methods for querying leaderboard definitions and records. Leaderboards are used to rank players based on their stats.

## Methods

### get_leaderboard_definitions_async

```gdscript
func get_leaderboard_definitions_async() -> Array
```

Returns an Array of Dictionary of all the leaderboards.

<ResponseField name="return" type="Array">
  Array of leaderboard definition dictionaries. Each leaderboard has the following data:
  - `leaderboard_id`: String - unique id
  - `stat_name`: String - name of stat used to rank the leaderboard
  - `aggregation`: EOS.Leaderboards.LeaderboardAggregation - aggregation used to sort the leaderboard
  - `start_time`: int - unix timestamp for the start time or `EOS.Leaderboard.LEADERBOARD_TIME_UNDEFINED`
  - `end_time`: int - unix timestamp for the end time or `EOS.Leaderboard.LEADERBOARD_TIME_UNDEFINED`
</ResponseField>

**Example:**

```gdscript
var leaderboards = await HLeaderboards.get_leaderboard_definitions_async()
for leaderboard in leaderboards:
    print("Leaderboard: ", leaderboard.leaderboard_id)
    print("Stat: ", leaderboard.stat_name)
    print("Aggregation: ", leaderboard.aggregation)
```

### get_leaderboard_records_async

```gdscript
func get_leaderboard_records_async(leaderboard_id: String) -> Array
```

Returns an Array of Dictionary of top records in the leaderboard.

<ParamField path="leaderboard_id" type="String" required>
  The unique identifier of the leaderboard to query
</ParamField>

<ResponseField name="return" type="Array">
  Array of leaderboard record dictionaries. Each record has the following data:
  - `leaderboard_id`: String - unique id
  - `user_id`: String - product user id of user associated with this record
  - `rank`: int - sorted position in the leaderboard
  - `score`: int - leaderboard score
  - `user_display_name`: String - latest display name for the user or empty string if the user doesn't have one
</ResponseField>

**Example:**

```gdscript
var records = await HLeaderboards.get_leaderboard_records_async("global_score")
for record in records:
    print("Rank ", record.rank, ": ", record.user_display_name, " - ", record.score)
```

**Complete Usage Example:**

```gdscript
# Get all leaderboard definitions
var leaderboards = await HLeaderboards.get_leaderboard_definitions_async()
if leaderboards.is_empty():
    print("No leaderboards configured")
    return

print("Available leaderboards:")
for leaderboard in leaderboards:
    print("  - ", leaderboard.leaderboard_id, " (stat: ", leaderboard.stat_name, ")")

# Get records for the first leaderboard
var leaderboard_id = leaderboards[0].leaderboard_id
var records = await HLeaderboards.get_leaderboard_records_async(leaderboard_id)

print("\nTop scores for ", leaderboard_id, ":")
for i in range(min(10, records.size())):
    var record = records[i]
    print("  #", record.rank, ": ", record.user_display_name, " - ", record.score, " points")
```

<Note>
  Leaderboards must be configured in the Epic Developer Portal before they can be queried. Each leaderboard is based on a stat that you've defined.
</Note>

<Tip>
  Use `HStats.ingest_stat_async()` to update player stats, which will automatically update their leaderboard rankings.
</Tip>
