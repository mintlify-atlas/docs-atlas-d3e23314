The Lobby Interface manages multiplayer lobbies for matchmaking and session management.

## Overview

The Lobby Interface provides:

- Lobby creation and management
- Lobby search and discovery
- Member management (kick, promote, mute)
- Lobby invites
- RTC (Real-Time Communication) room integration
- Lobby attributes and metadata

All methods are called on the `IEOS` singleton instance.

## Methods

### lobby_interface_create_lobby

```gdscript
IEOS.lobby_interface_create_lobby(options: RefCounted) -> void
```

Creates a new lobby.

**Parameters:**
- `options` - Dictionary containing:
  - `local_user_id` - Product User ID
  - `max_lobby_members` - Maximum number of members
  - `permission_level` - EOS_ELobbyPermissionLevel
  - `presence_enabled` - Boolean for presence
  - `allow_invites` - Boolean
  - `bucket_id` - Bucket ID string
  - `disable_host_migration` - Boolean
  - `enable_rtc_room` - Boolean for voice chat
  - `rtc_room_join_action_type` - EOS_ELobbyRTCRoomJoinActionType
  - `local_rtc_options` - Optional RTC options
  - `lobby_id` - Optional custom lobby ID
  - `enable_join_by_id` - Boolean
  - `rejoin_after_kick_requires_invite` - Boolean
  - `allowed_platform_ids` - Optional array of platform IDs
  - `client_data` - Optional user data

**Callback Signal:** `lobby_interface_create_lobby_callback`

## Example Usage

```gdscript
# Create a lobby
var create_options = {
    "local_user_id": product_user_id,
    "max_lobby_members": 4,
    "permission_level": EOS_ELobbyPermissionLevel.EOS_LPL_PUBLICADVERTISED,
    "presence_enabled": true,
    "allow_invites": true,
    "bucket_id": "MyGame:Region1",
    "enable_rtc_room": true,
    "rtc_room_join_action_type": 0
}

IEOS.lobby_interface_create_lobby_callback.connect(_on_lobby_created)
IEOS.lobby_interface_create_lobby(create_options)

func _on_lobby_created(data: Dictionary):
    if data.result_code == EOS_EResult.EOS_Success:
        print("Lobby created: ", data.lobby_id)
```
