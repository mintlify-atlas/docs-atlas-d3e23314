The UI Interface manages Epic Games Overlay interactions and social features.

## Overview

The UI Interface allows:

- Showing/hiding the friends overlay
- Displaying block and report player dialogs
- Managing overlay settings and input
- Configuring notification preferences

All methods are called on the `IEOS` singleton instance.

## Methods

### ui_interface_show_friends

```gdscript
IEOS.ui_interface_show_friends(options: RefCounted) -> void
```

Shows the Epic Games friends overlay.

**Parameters:**
- `options` - Dictionary containing:
  - `local_user_id` - Epic Account ID
  - `client_data` - Optional user data

**Callback Signal:** `ui_interface_show_friends_callback`

### ui_interface_hide_friends

```gdscript
IEOS.ui_interface_hide_friends(options: RefCounted) -> void
```

Hides the Epic Games friends overlay.

**Callback Signal:** `ui_interface_hide_friends_callback`

### ui_interface_get_friends_visible

```gdscript
IEOS.ui_interface_get_friends_visible(options: RefCounted) -> bool
```

Checks if the friends overlay is currently visible.

**Parameters:**
- `options` - Dictionary containing:
  - `local_user_id` - Epic Account ID

**Returns:** Boolean indicating visibility

### ui_interface_get_friends_exclusive_input

```gdscript
IEOS.ui_interface_get_friends_exclusive_input(options: RefCounted) -> bool
```

## Example Usage

```gdscript
# Show friends overlay
var show_options = {
    "local_user_id": epic_account_id
}

IEOS.ui_interface_show_friends_callback.connect(_on_overlay_shown)
IEOS.ui_interface_show_friends(show_options)

func _on_overlay_shown(data: Dictionary):
    if data.result_code == EOS_EResult.EOS_Success:
        print("Friends overlay shown")

# Show block player dialog
var block_options = {
    "local_user_id": epic_account_id,
    "target_user_id": target_epic_id
}
IEOS.ui_interface_show_block_player(block_options)

# Set notification location
var result = IEOS.ui_interface_set_display_preference({
    "notification_location": EOS_UI_ENotificationLocation.EOS_UNL_TopRight
})
```
