## Overview

HAchievements provides methods to query achievement definitions, get player achievement progress, and unlock achievements.

## Signals

### achievement_unlocked

```gdscript
signal achievement_unlocked(data: Dictionary)
```

Emitted when an achievement is unlocked.

**Data dictionary keys:**
- `achievement_id: String` - The achievement ID
- `unlock_time: int` - Unix timestamp when unlocked
- `user_id: String` - Product user ID of the user

## Methods

### get_all_achievements_async

```gdscript
func get_all_achievements_async() -> Array[HAchievementData]
```

Returns an array of [HAchievementData](#hachievementdata) representing all the achievements in the game.

**Returns:** Array of HAchievementData objects

**Example:**
```gdscript
var achievements = await HAchievements.get_all_achievements_async()
for achievement in achievements:
    print(achievement.achievement_id, ": ", achievement.get_display_name())
```

### get_player_achievements_async

```gdscript
func get_player_achievements_async() -> Array[HAchievementData]
```

Returns an array of [HAchievementData](#hachievementdata) representing all the player's achievements with current progress.

**Returns:** Array of HAchievementData objects with progress information

**Example:**
```gdscript
var player_achievements = await HAchievements.get_player_achievements_async()
for achievement in player_achievements:
    print(achievement.get_display_name())
    print("Progress: ", achievement.progress, "%")
    if achievement.is_unlocked():
        print("Unlocked at: ", achievement.unlock_time)
```

## Usage Example

```gdscript
extends Node

func _ready():
    # Connect to achievement unlocked signal
    HAchievements.achievement_unlocked.connect(_on_achievement_unlocked)
    
    # Load all achievements
    var all_achievements = await HAchievements.get_all_achievements_async()
    print("Total achievements: ", all_achievements.size())
    
    # Check player progress
    var player_achievements = await HAchievements.get_player_achievements_async()
    for achievement in player_achievements:
        if achievement.is_unlocked():
            print("✓ ", achievement.get_display_name())
        else:
            print("✗ ", achievement.get_display_name(), " (", achievement.progress, "%)")

func _on_player_won_first_match():
    # Unlock achievement when player wins first match
    await HAchievements.unlock_achievement_async("FIRST_VICTORY")

func _on_achievement_unlocked(data: Dictionary):
    print("Achievement unlocked: ", data.achievement_id)
    # Show achievement notification UI
```

## Best Practices

<Note>
Achievements must be configured in the Epic Games Developer Portal before they can be used in your game.
</Note>

1. **Cache achievements**: Call `get_all_achievements_async()` once at startup and cache the results
2. **Monitor progress**: Use the `stats` array in HAchievementData to track achievement progress
3. **Handle unlock events**: Connect to the `achievement_unlocked` signal to show notifications
4. **Batch unlocks**: Use `unlock_achievements_async()` to unlock multiple achievements efficiently
