## Overview

HLobbies provides methods to create lobbies, search for existing lobbies, and join lobbies. It returns [HLobby](#hlobby) objects that can be used to manage lobby state and members.

## Properties

### max_search_results

```gdscript
var max_search_results: int = 25
```

The maximum number of lobbies to return in search calls (except for `create_search` and `search_async`).

### presence_enabled

```gdscript
var presence_enabled: bool = true
```

Whether to show the lobby in the user's presence.

### local_rtc_options

```gdscript
var local_rtc_options: Dictionary
```

Local RTC options for the lobby. You can override any of the following keys:
- `flags: int` - A bitwise-or union of `EOS.RTC.JoinRoomFlags`
- `use_manual_audio_input: bool`
- `use_manual_audio_output: bool`
- `local_audio_device_input_starts_muted: bool`

Defaults to:
```gdscript
{
    flags = EOS.RTC.JoinRoomFlags.EnableDataChannel
}
```

### search_by_attribute_async

```gdscript
func search_by_attribute_async(attributes) -> Array[HLobby]
```

Search for public lobbies based on attributes.

**Parameters:**
- `attributes`: Dictionary or Array of dictionaries with keys:
  - `key: String` - the key of the attribute
  - `value: Variant` - the value of the attribute
  - `comparison: EOS.ComparisonOp` - Type of comparison to make (default is `EOS.ComparisonOp.Equal`)

**Returns:** Array of [HLobby](#hlobby) objects or `null` on failure

**Example:**
```gdscript
# Search for lobbies with game_mode = "deathmatch"
var lobbies = await HLobbies.search_by_attribute_async({
    key = "game_mode",
    value = "deathmatch"
})

# Search with multiple attributes (implicit AND)
var lobbies = await HLobbies.search_by_attribute_async([
    {key = "game_mode", value = "deathmatch"},
    {key = "map", value = "desert"},
    {key = "max_players", value = 4, comparison = EOS.ComparisonOp.LessThanOrEqual}
])
```

## Usage Example

```gdscript
extends Node

var current_lobby: HLobby

func create_and_host_lobby():
    var opts = EOS.Lobby.CreateLobbyOptions.new()
    opts.max_lobby_members = 4
    opts.permission_level = EOS.Lobby.LobbyPermissionLevel.Publicadvertised
    opts.bucket_id = "main_lobby"
    
    current_lobby = await HLobbies.create_lobby_async(opts)
    if current_lobby:
        current_lobby.lobby_updated.connect(_on_lobby_updated)
        current_lobby.kicked_from_lobby.connect(_on_kicked)
        
        # Add custom attributes
        current_lobby.add_attribute("game_mode", "deathmatch")
        current_lobby.add_attribute("map", "desert")
        await current_lobby.update_async()
        
        print("Lobby created: ", current_lobby.lobby_id)

func find_and_join_lobby():
    var lobbies = await HLobbies.search_by_bucket_id_async("main_lobby")
    
    if lobbies and lobbies.size() > 0:
        current_lobby = await HLobbies.join_async(lobbies[0])
        if current_lobby:
            current_lobby.lobby_updated.connect(_on_lobby_updated)
            current_lobby.kicked_from_lobby.connect(_on_kicked)
            print("Joined lobby: ", current_lobby.lobby_id)

func _on_lobby_updated():
    print("Lobby updated: ", current_lobby.members.size(), " members")
    for member in current_lobby.members:
        print("  - ", member.product_user_id)

func _on_kicked():
    print("Kicked from lobby")
    current_lobby = null

func send_message_to_lobby(message: String):
    if current_lobby and current_lobby.rtc_room_connected:
        var data = {"type": "chat", "message": message}
        current_lobby.rtc_send_data(data)
```

## Best Practices

1. **Use bucket IDs**: Organize lobbies into buckets for easier searching and filtering
2. **Set appropriate permissions**: Choose the right permission level for your lobby type
3. **Monitor lobby updates**: Connect to `lobby_updated` signal to keep UI in sync
4. **Clean up**: Always call `leave_async()` or `destroy_async()` when done with a lobby
5. **Handle search results**: Check if search results are null or empty before accessing
6. **Use attributes**: Store custom lobby metadata using attributes for flexible searching

<Warning>
Always check if a lobby is valid before performing operations on it using `is_valid()`.
</Warning>
